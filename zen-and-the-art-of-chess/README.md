## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Tech Stack

- **React 18** with TypeScript
- **Vite** for fast development
- **TailwindCSS** for styling
- **Zustand** for state management
- **react-chessboard** for the board UI
- **chess.js** for move validation
- **Stockfish WASM** for engine analysis

## Design Philosophy

The app follows a minimal, elegant aesthetic:
- Dark theme with slate colors
- Glass-morphism effects
- Serif typography for teachings
- Subtle gold accents
- Responsive design

## Usage

1. Start with Day 1 on the Daily page
2. Read the teaching and commentary
3. Complete the meditation
4. Do the daily exercise
5. Track your mental state with the Tilt Tracker
6. Practice puzzles in Pattern Training
7. Play mindfully in Calm Play mode
